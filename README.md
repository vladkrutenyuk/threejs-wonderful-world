# 3D Interactive World Map
#### Web / Typescript / Three.js
---
### Run
1. `npm install`
2. `npm run dev`
3. enjoy at 'http://localhost:3000/'

### Features
#### 1. Camera orbit controls
It includes pan, zoom, rotation.
#### 2. Map controls
E - Zoom In

Q - Zoom Out

W, A, S, D - Move
#### 3. Markers on map
#### 4. Magic cursor/crosshair
